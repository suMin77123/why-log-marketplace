# Why Log Marketplace

Plugin marketplace for [why-log](https://github.com/suMin77123/why-log).

## Installation

Run the following commands inside Claude Code:

```
/plugin marketplace add suMin77123/why-log-marketplace
/plugin install why-log@why-log-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [why-log](https://github.com/suMin77123/why-log) | Automatic decision logging for AI coding agents |
